== Description ==
Add new menu items to the WordPress Navigation Menus interface for login, logout, registration, and more (with or without a current URL redirect).

== Installation ==
= General Requirements =
* PHP: 7.4 or newer
* bbPress 2.6.2 or newer
* WordPress: 5.8 or newer

= PHP Notice =
* The plugin doesn't work with PHP 7.3 or older versions.

= Basic Installation =
* Upload `navmenu-enhanced` folder to the `/wp-content/plugins/` directory.
* Activate the plugin through the 'Plugins' menu in WordPress.
* New menu items will be available in the WordPress Menus interface.
